hexo-setup

环境搭建

安装 node.js

1 nodejs.org下载安装，安装后使用 node -v 和 nmp -v 确定都有成功被安装

2 npm install -g cnmp –registry=https://registry.npm.taobao.org

安装cnpm替代npm使用国内镜像源，cnpm -v确定有安装成功

安装 hexo-cli

cnpm install -g hexo-cli 安装hexo， hexo -v 验证安装成功

创建 Hexo 博客

1 本地创建文件夹 blog

2 在 blog 文件夹下 hexo init 来初始化 hexo

3 hexo s 在本地启动 hexo 服务，localhost:4000 访问 hexo 博客，用于本地显示博客样式用

4 hexo n “first blog name” 新建一篇博客，即新建 first blog name.md 到 source/_posts/下，可以用 typora来编辑 first blog name.md 编辑自己的博客

5 hexo clean 先清理一下，hexo g 生成博客，hexo s 本地查看博客

6 github 创建仓库，仓库名字为：githubname.github.io

7 cnpm install –save hexo-deployer-git

8 在 _config.yml 最后修改为

deploy:

type: git

repo: git@github.com:githubname/githubname.github.io.git (http可能会失败)

branch: master

9 hexo d 将 hexo 部署到github上

10 通过 githubname.github.io 即可访问博客
